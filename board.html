Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML: It is the block of all html elements,comments,text inside a parent div/tag.
2. innerText: It is the visible content of an html element.
3.textContent: All text that are visible and hidden and strips out the tag.

Question-3: What is event delegation in the DOM?

Event delegation let us handle events by putting a single event listener to a parent element instead of attaching to each child.

Question-4: What is event bubbling in the DOM?

It is the propagation of an event triggered from event source to the document root.
